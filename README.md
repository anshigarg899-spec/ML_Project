#  Experiment 1: Car Dataset Preprocessing

##  Objective
To perform data preprocessing on the car dataset to make it suitable for machine learning models.

##  Dataset
- File Name: car_dataset.csv
- Description: Contains information about cars such as price, features, and specifications.

##  Steps Performed

### 1. Data Loading
Loaded dataset using pandas.

### 2. Data Understanding
Checked structure using info() and describe().

### 3. Handling Missing Values
Missing values were handled using mean or by removing null rows.

### 4. Removing Duplicates
Duplicate rows were removed to improve data quality.

### 5. Encoding Categorical Data
Converted categorical columns into numerical using encoding.

### 6. Feature Scaling
Applied StandardScaler to normalize data.

### 7. Train-Test Split
Split dataset into training and testing sets.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn

##  Output
The dataset is cleaned and ready for machine learning models.
